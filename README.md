# Pyinecraft

Pyinecraft is a simple minecraft clone made with Python.

<img src="iii.png" alt="iii"/>

## Bugs

You cannot break the blocks in the ground because, if you even break a block the hole (where the block was) will follow you.
   
## How To Play:

1 = Grass <br>
2 = Dirt <br>
3 = Stone <br>
4 = Cobblestone <br>
5 = Sand <br>
6 = Log <br>
7 = Planks <br>
8 = Obsidian <br>
9 = Ice <br>
